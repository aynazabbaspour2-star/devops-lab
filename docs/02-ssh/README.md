# SSH Authentication

## Goal

Configure secure SSH authentication between:

- Laptop
- Debian VPS
- GitHub

## Completed

- SSH key authentication
- Passwordless login
- Non-root user
- GitHub SSH authentication

## Commands Used

```bash
ssh-keygen
ssh-copy-id
ssh -T git@github.com
git clone git@github.com:aynazabbaspour2-star/devops-lab.git
```

## Notes

SSH authentication is working successfully between my local machine, VPS, and GitHub.
